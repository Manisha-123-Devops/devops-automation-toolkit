# DevOps Automation Toolkit

## Overview
This repository contains operational automation scripts designed to support AWS infrastructure, Kubernetes clusters, CI/CD workflows, and production monitoring.

## Areas Covered
- AWS infrastructure validation
- S3 backup verification
- Kubernetes cluster health checks
- Pod stability monitoring
- Application endpoint validation
- Docker cleanup automation

## Technologies Used
- Bash scripting
- Python automation
- AWS CLI
- kubectl
- psutil library

## Use Cases
- Production support automation
- Incident troubleshooting
- Operational validation
- CI/CD environment maintenance

## Best Practices
- Scripts designed for repeatable execution
- No hardcoded credentials
- Intended for integration into cron jobs or CI/CD pipelines
